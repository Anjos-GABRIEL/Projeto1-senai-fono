# Projeto Fonoaudiologia - Instituto Ouvir e Brincar

Este projeto é uma Landing Page premium desenvolvida para o **Instituto Ouvir e Brincar**, focada em serviços de fonoaudiologia. O site foi construído utilizando tecnologias modernas de desenvolvimento web para proporcionar uma experiência de usuário fluida, interativa e profissional.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização avançada com animações, gradientes e design responsivo.
- **JavaScript**: Interatividade, efeitos de scroll e lógica de interface.
- **Google Fonts**: Tipografia moderna (Inter e Outfit).
- **Lucide Icons**: Ícones minimalistas e elegantes.

## ✨ Funcionalidades

- **Design Premium**: Visual moderno com glassmorphism e micro-animações.
- **Responsividade**: Totalmente adaptado para dispositivos móveis, tablets e desktops.
- **Seção de Serviços**: Apresentação detalhada dos tratamentos oferecidos.
- **Interatividade**: Menu responsivo e animações ao rolar a página (AOS).
- **SEO Otimizado**: Estrutura focada em mecanismos de busca.

## 📁 Estrutura do Projeto

- `index.html`: Arquivo principal contendo a estrutura do site.
- `style.css`: Estilização completa do projeto.
- `script.js`: Lógica de interatividade e animações.
- `dra_laura_silva.png`: Imagem profissional da especialista.

## 🛠️ Como Executar

Para visualizar o projeto localmente:

   ```bash
   git clone https://github.com/Anjos-GABRIEL/ProjetoFonoinstituoouvirebrincar.git
   ```
