# 🏥 Dr. Laura Fono - Landing Page Premium

Este repositório contém o código-fonte da Landing Page institucional da **Dra. Laura Fono**. O projeto foi concebido com foco em alta performance, design premium e acessibilidade, servindo como a porta de entrada digital para pacientes que buscam excelência em fonoaudiologia.

## 💎 Design e Experiência do Usuário (UX/UI)

A aplicação utiliza as melhores práticas de design moderno para transmitir confiança e profissionalismo:
- **Aestética Premium**: Uso de gradientes suaves, *glassmorphism* e tipografia moderna (Outfit & Inter).
- **Micro-interações**: Feedback visual dinâmico em botões e elementos interativos.
- **Responsividade Total**: Layout adaptável para smartphones, tablets e desktops (Mobile-First).
- **Animações Fluidas**: Implementação de efeitos de surgimento ao rolar a página (*Scroll Animations*).

## 🛠️ Tecnologias e Arquitetura

O projeto foi construído utilizando tecnologias fundamentais para garantir leveza e SEO otimizado:

- **Estrutura**: HTML5 Semântico para melhor indexação em motores de busca.
- **Estilização**: CSS3 Vanilla com variáveis para fácil manutenção e temas.
- **Lógica**: JavaScript ES6+ para manipulação de DOM e interatividade.
- **Ícones**: Boxicons para uma interface limpa e intuitiva.
- **Fontes**: Google Fonts API para carregamento otimizado de tipografia.

## 🚀 Como Iniciar o Projeto

### Pré-requisitos
Embora seja um projeto estático, recomendamos o uso de um servidor local para melhor experiência com as animações e assets:
- [Node.js](https://nodejs.org/) (Opcional, para scripts de dev)

### Passo a Passo

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Anjos-GABRIEL/Projeto1-senai-fono.git
   ```

2. **Navegue até a pasta:**
   ```bash
   cd "01 Landing Page"
   ```

3. **Inicie o servidor de desenvolvimento:**
   *Se você tiver Node.js:*
   ```bash
   npm install
   npm run dev
   ```
   *Ou apenas abra o `index.html` diretamente no navegador.*
